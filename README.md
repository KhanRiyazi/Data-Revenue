## 📖 Overview

**RIZWAN · Build & Earn with Data** is a single-page application that serves as a **portfolio, learning hub, and client acquisition tool** all in one. It showcases:

- ✅ **Live projects** with GitHub and deployment links
- ✅ **Revenue‑focused interview Q&A** across Python Backend, Data Science, and Business Analytics
- ✅ **Curated learning resources** with detailed "what you'll learn" sections
- ✅ **Professional services** with measurable impact metrics
- ✅ **Trust signals** (testimonials, stats, social proof)

The design is **dark‑themed**, **fully responsive**, and built with a **glass‑morphism** aesthetic that feels premium and modern.

### 🎯 Purpose

This dashboard is designed to:

1. **Showcase your technical expertise** through live projects
2. **Demonstrate your business acumen** through revenue-focused content
3. **Build trust** with testimonials and social proof
4. **Generate leads** through a functional contact form
5. **Share knowledge** through curated resources and interview Q&A
6. **Turn data into revenue** by bridging technical skills with business outcomes

## ✨ Features

### 🎯 Core Features

| Feature | Description |
|---------|-------------|
| **Sticky Glass Header** | Elegant navigation with blur effect, stays at top while scrolling |
| **Scroll Progress Bar** | Visual indicator showing how far you've scrolled |
| **Hero Section** | Value proposition with clear call-to-action buttons |
| **Trust Stats** | Key metrics (repositories, ML models, delivery rate) |
| **Services** | Three service cards with measurable impact tags |
| **Premium Project Cards** | Image-first design with dual action buttons (Code/Live) |
| **Project Filtering** | Filter projects by tag (FastAPI, ML, Design) |
| **Suspensive Learning** | Click-to-reveal Q&A with nested answer popups |
| **Curriculum Accordion** | Expandable sections for Python, Data Science, Business |
| **Dashboard Extras** | Model metrics, recent feed, monthly activity chart |
| **Blog Section** | Blog post previews with read-more links |
| **Curated Resources** | Filterable learning resources with detailed modals |
| **Testimonials** | Client feedback with project context |
| **Contact Form** | Integrated with EmailJS for real message delivery |
| **Back to Top** | Floating button appears after scrolling |

### 🎨 Design Features

- 🎭 **Dark Theme** with gold and violet accents
- 🪞 **Glass‑morphism** effects on cards and modals
- ✨ **Smooth animations** and micro‑interactions
- 📱 **Fully responsive** (mobile, tablet, desktop)
- 🔤 **Premium typography** with Inter font
- 🎯 **FontAwesome icons** throughout
- 🌈 **Gradient accents** for visual hierarchy
- 💫 **Hover effects** for interactive feedback
- 🎪 **Modal popups** for detailed content viewing

## 🛠️ Tech Stack

| Technology | Purpose |
|------------|---------|
| **HTML5** | Structure & semantic markup |
| **CSS3** | Custom styles with CSS variables, animations, glass‑morphism |
| **Vanilla JavaScript** | Interactive features, modals, filtering, animations |
| **EmailJS** | Contact form email delivery |
| **FontAwesome 6** | Icon library |
| **GitHub Pages / Render** | Hosting & deployment |

### Why Vanilla JavaScript?

- ✅ **No build tools** required
- ✅ **Zero dependencies** – just one file
- ✅ **Lightweight** – loads instantly
- ✅ **Easy to customize** – everything is in one place
- ✅ **SEO friendly** – fully semantic HTML
- ✅ **Faster load times** – no framework overhead
